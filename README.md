# official-dezhou-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常为实验室内部项目、工具或演示的静态版本，不针对任何特定域名或网站。

## 项目简介

该仓库作为一个静态 HTML 页面的集合与发布点。每个页面均为独立文件，可能包含样式、脚本或资源引用。仓库不承载动态后端功能，所有内容均为前端静态实现。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：各独立 HTML 页面存放于此。每个文件可独立运行。
- `assets/`：页面共享的静态资源，使用时请确保路径正确。

## 页面归档说明

- 每个 HTML 页面应为完整、自包含的文档（可引用外部资源）。
- 页面命名建议使用英文小写，以连字符或下划线分隔。
- 如需引用仓库内资源，请使用相对路径，避免硬编码外部链接。
- 新增页面时，请同步更新本文件或添加简要说明。

## 维护说明

- 本仓库主要面向内部归档与分发，不承诺长期在线服务。
- 提交新页面或更新时，请确保内容不包含敏感信息或外部依赖。
- 若页面依赖特定外部库，建议在页面内注明版本或来源。
- 欢迎通过 Issue 或 Pull Request 提出改进建议。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 如需部署，可将整个仓库内容上传至任意静态托管服务。
